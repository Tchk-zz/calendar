# Календарь 2026

Простой интерактивный календарь на 2026 год, созданный с использованием HTML, CSS и JavaScript.

## Описание файлов

- **index.html** — основная структура страницы: контейнер календаря, заголовок с названием месяца и года, кнопки переключения месяцев, таблица с днями недели. Подключает файл стилей и скрипт.
- **style.css** — отвечает за внешний вид календаря: новогодний фон, шрифт Roboto, стили для выходных (красноватый фон) и праздничных дней (ярко-красный), эффекты hover, современный дизайн.
- **script.js** — логика работы календаря: генерация дней месяца, определение первого дня недели (2026 начинается в четверг), переключение месяцев, выделение выходных, праздничных дней и текущего дня.

## Подключение файлов в index.html

- Файл `style.css` подключается в строке 7:  
  `<link rel="stylesheet" href="style.css">`
- Файл `script.js` подключается в строке 23:  
  `<script src="script.js"></script>`

## Этапы создания

1. Создан репозиторий Calendar на GitHub.
2. Добавлены файлы: index.html, style.css, script.js, README.md и фон background.png.
3. Реализована структура календаря (HTML).
4. Оформлен внешний вид, добавлен новогодний фон, изменён шрифт на Roboto (подключён через Google Fonts в CSS).
5. Написана логика на JavaScript: генерация дат, переключение месяцев, выделение выходных и праздничных дней.
6. Включены GitHub Pages для публикации.

## Особенности оформления

- Новогодний фон (background.png)
- Шрифт: Roboto
- Выходные дни (сб, вс) — светло-красный фон
- Праздничные дни — ярко-красный фон с белым текстом
